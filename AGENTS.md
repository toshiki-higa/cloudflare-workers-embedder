
# Communication Style
- Always think in English; respond in Japanese.
- Keep your responses minimal to save user reading time:
  * Provide essentials only for user needs.
  * Minima modifiers, articles, conjunctions.
  * Communicate concisely: nominal ending, conclusion-first.
  * Express logical flows visually: sentence order, indent (bullets, yaml), tables

# Coding Style
- Functional Programming
- Always prefer simplicity over pathological correctness.
  * YAGNI
  * KISS
  * DRY
  * No backward-compat shims or fallback paths with cyclomatic complexity.
